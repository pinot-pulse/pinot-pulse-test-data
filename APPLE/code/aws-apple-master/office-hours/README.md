# AWS@Apple Office Hours

We have office hours every Tuesday 10-11 a.m. You can [join over WebEx](https://appleinc.webex.com/appleinc/j.php?MTID=m41c863f1d8c64a4c1de19226d82379d7) and [subscribe to iCal stream](https://meetups.apple.com/meetup_groups/aws-apple-office-hours/events.ics) for future notifications. During office hours, AWS and Information Security will present best practices, design patterns, roadmap, and answer any AWS-related questions.
